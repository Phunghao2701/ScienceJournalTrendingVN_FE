## Sprint Goal

Integrate the Paper VN Article Discovery page with the verified backend contract, remove incorrect API calls and N+1 behavior, and make the UI accurately represent Vietnamese-university article results.

## Preconditions

- Preserve branch `ngoc/feature/trending-lens-style` and dirty worktree.
- Read BE handoff/API examples before editing FE contracts.
- Do not modify core Trending VN dashboard or `/trending-vn` analytics endpoints.
- Do not install new dependencies.

## Risks and Guards

- **BE contract drift**: read backend handoff before implementation.
- **Large page merge conflicts**: extract focused hooks/components incrementally.
- **Incorrect analytics filters**: derive list and analytics params from one normalized helper.
- **Unsafe title rendering**: sanitize or strip XML; never trust raw HTML.
